#

1. 18款15寸macbookpro

之前的电脑在学校用了很长的时间，多次损坏最后换了台18款macbookpro。

主要是喜欢macbookpro的Retina屏幕，确实舒服，但是看久了还是有点眼睛难受。

还一个原因就是macbookpro的操作系统macos，类unix的体验让我十分爽快。

但是，买macbookpro的后续东西比较繁琐，还要买各种转接线（劣质的不推荐），或者需要的还有键盘膜，屏幕膜，电脑保护壳。

2. macos软件

  1. 关于macos的小工具（基本必备的软件）

图中为小工具必备软件

  1. notability和印象笔记

  1. Chrome浏览器（因为性能快速和万能的插件）

  2. finalcutpro x和motion

    剪辑视频

  3. sketch或者axure pro
    做原型

  4. eagle

  5. IINA播放器（开源免费功能强大）

  6. Reeder3（万物皆可RSS）

    由于现在RSS源不太好找，推荐下面这个教程进行设置，方便很多。

    https://docs.rsshub.app/#%E9%B8%A3%E8%B0%A2

  7. 各种编程相关软件（IDEA/Pycharm/xcode/anaconda/postman等等）

  8. 幕布（做大纲）

  9. things3

    做事情提醒事项，还有很多扩展事项，设备同步很方便。

  10. duet（屏幕扩展）

    比如ipad下载duet软件，mac下运行duet软件，ipad就会变成扩展屏幕，还可以设置清晰度/刷新率/镜像屏幕等等。

  11. SecureCRT（远程ssh软件）

  12. Typora（简洁好用免费的markdown软件）

  13. maipo（微博客户端，很多实用的功能，很好用）

  14. pap.er和iwall（前者是免费壁纸软件，后者是动态桌面壁纸软件）

  15. 网易云音乐/QQ音乐/Noizio（第三个是环境音，很舒服的哦）

  16. Crossover（macos上运行windows软件）

  17. Parallels / vmware / vitualbox（虚拟机自然少不了了，哪个好看个人喜好习惯）

  18. CleanMyMac X / DaisyDisk / CleanMyDrive 2（清理工具）

  19. Marginnote 3 （很棒的pdf学习工具，还有脑图和复习功能等等，提高学习效率）

  20. Downie3（结合插件，超强下载功能）

  21. ipic和upic（前者图床，后者压缩）

  22. 坚果云

  23. keynote和office全家桶

还有一些软件就是个人需求而定了，就不做介绍了。这些基本上就是必备的软件，当然呢正版效果更好，所以肯定需要在软件上花上一些资金来体验高效生活效率的乐趣。
