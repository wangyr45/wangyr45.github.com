macos终端配置mysql

February 28 2019

  1. 打开终端，输入 open .bash_profile

  2. 在文本上追加：

    export PATH=${PATH}:/usr/local/mysql/bin

    mysql=/usr/local/mysql/bin/mysql

    mysqladmin=/usr/local/mysql/bin/mysqladmin

  3. 保存并退出

  4. 在终端上输入 source .bash_profile 生效

  5. 输入mysql -u root -p

  6. 完毕。
